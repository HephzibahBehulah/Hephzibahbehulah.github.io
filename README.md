## 🎨 Features

### ✨ Design & UX
- **Modern, responsive design** - Optimized for all devices
- **Dark/Light mode** - User preference with system detection
- **Smooth animations** - AOS.js and Animate.css
- **Accessibility** - ARIA labels, semantic HTML, keyboard navigation
- **GDPR compliant** - No tracking cookies unless disclosed

### 🔧 Technical Features
- **Modular code structure** - Easy to maintain and update
- **SEO optimized** - Meta tags, sitemap, structured data
- **Fast loading** - Minified assets, lazy loading, CDN
- **Lighthouse score** > 90 for all categories
- **GitHub Pages** hosting with custom domain

1. **Home** - Hero section with mission statement
2. **About** - Bio, experience, skills, certifications
